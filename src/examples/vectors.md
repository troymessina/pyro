# Vectors

This example creates a static scene with three unit vectors labeled x, y, z each positioned and colored distinctly.

## What You'll See

- **x unit vector** — Green arrow at the origin (length 1) pointed along the x-axis
- **y unit vector** — Blue arrow at the origin (length 1) pointed along the y-axis
- **z unit vector** — Red arrow at the origin (length 1) pointed along the z-axis

## Concepts Demonstrated

- `arrow(pos, axis, color)` — Directional vectors
- `label( pos, text, box, opacity )`
